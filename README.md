# 🎯 Feud Bilingüe

A bilingual Family Feud-style trivia game built with React that helps players learn Spanish through gameplay. Guess survey answers in **English or Spanish** — both work.

🔗 **[Play it live](https://Dpalm88.github.io/bilingual-feud-game)**

---

## What It Does

- 3-round Family Feud format with a classic game-show aesthetic
- Every answer revealed in both English and Spanish side-by-side
- Guess in either language — fuzzy matching handles typos and accents
- Round 1 questions are **dynamically generated by Claude AI** (Anthropic API) with a fresh theme each game
- Rounds 2 & 3 pull from a curated question bank
- 3 strikes ends the round; score accumulates across all rounds

| Layer | Tech |
|---|---|

---

## API Integrations

**Anthropic Claude API** — Generates unique trivia questions dynamically at runtime. Each new game hits the API with a random theme prompt and parses a structured JSON response containing bilingual questions and ranked answers with point values.

**MyMemory Translation API** — Free REST translation endpoint used for real-time language support. No API key required, supports English ↔ Spanish with automatic language detection.

## Features in Detail

- **Fuzzy answer matching** — Levenshtein distance algorithm tolerates typos up to 2 character edits
- **Accent normalization** — Strips diacritics so `refrigerador` and `refrigeradór` both match
- **Flip tile animations** — CSS 3D transforms on answer reveal with staggered timing
- **Shake animation** — Input shakes on wrong guess with red strike indicator
- **Responsive layout** — Playable on mobile and desktop
